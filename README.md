# Description
   hubot plugin to write like an idiot
   This plugin will rewrite your own message on rocketchat

# Configuration
   None

# Commands
   .dau <mytext> - Write like an idiot

# Example
```
.dau this is crazy
AEH , TIHS IS CRAZY !!!!!!!!!!?!!!!?!!?!1
```

# Publish
```
npm publish --access=public
```
